# X-Technologien

Code und Materialien für die XSLT-Schulung an der OBVSG.

## Setup
Die einzige technische Voraussetzung ist, dass Java auf dem System installiert ist. Alles andere sollte in diesem Projekt enthalten sein.

### Das Projekt herunterladen

Wenn Sie `git` verwenden, klonen Sie dieses Projekt einfach. Anonsten können Sie es herunterladen, indem sie auf das Pfeilchen am großen, grünen "<> Code"-Button klicken und dort "Download ZIP" auswählen. Entpacken Sie die Datei an einem Ort Ihrer Wahl.
